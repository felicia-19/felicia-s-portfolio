TES Hypomethylation as a Breast Cancer Biomarker (ISEF 2024)

Independent research · ISEF Grand Award · 2024

Analyzed bisulfite sequencing data and extracted cytosine-level methylation metrics across gene regions. Identified a statistically significant hypomethylation pattern at transcription ending sites (TES) as a candidate biomarker with potential for minimally invasive detection via circulating tumor DNA.

poster / writeup / code / media

Single-Cell Time-Lapse Tracking for Immunotherapy Evaluation (Digi.Bio)

Data Scientist · Netherlands · industry · Gap year

Built components for time-lapse single-cell tracking: graph-based cell ID propagation, split/merge handling during division, and preprocessing for noisy microscopy (background correction, filtering, adaptive thresholding). Worked with classical ML (random forests) and segmentation models (e.g., Cellpose, SAM).

demo / technical note

Automated Quality Control for Neonatal Dried Blood Spot Screening

Independent project · Computer Vision + ML · Ongoing

Developed a computer vision pipeline for DBS quality assessment as an alternative to subjective visual inspection. Combines feature extraction with CNN classification and a random forest model in a continuous training loop, with pilot deployment in Moldova’s Neonatal Screening Laboratory and additional trials planned.
